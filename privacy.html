Kebijakan Privasi

Terakhir diperbarui: 19 Maret 2026

1. Komitmen Privasi Kami

MeetingAI Pro dirancang dengan prinsip "Privacy by Design". Kami percaya bahwa data rapat Anda sangat sensitif dan tidak boleh meninggalkan perangkat Anda tanpa izin eksplisit.

2. Pemrosesan Data Lokal

Seluruh proses transkripsi suara dan ringkasan AI (Notulen) dilakukan secara lokal di perangkat Anda menggunakan teknologi Apple Speech Framework dan model AI on-device. Kami tidak mengirimkan rekaman suara atau teks transkrip Anda ke server kami.

3. Penggunaan Gemini AI (Opsional)

Jika Anda memilih untuk mengaktifkan fitur "Gemini Cloud AI" di pengaturan, teks transkrip akan dikirimkan ke Google melalui API untuk diproses. Ini hanya terjadi atas permintaan Anda secara manual dan menggunakan API Key pribadi Anda.

4. Keamanan Data

Data Anda tersimpan di database lokal yang terlindungi oleh sistem keamanan standar iOS/macOS (Sandboxing). Kami tidak memiliki akses ke data tersebut.

5. Kontak

Jika ada pertanyaan mengenai privasi ini, silakan hubungi tim dukungan kami melalui halaman bantuan.
